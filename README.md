# Issue Tracker

## 🚀 Project Setup Guide

---

## 1. Database Setup

Create the database using MySQL:

```bash
mysql -u root -p < database.sql
```

Or import `database.sql` using phpMyAdmin or MySQL Workbench.

---

## 2. Backend Setup

```bash
cd backend
npm install
```

Create `.env` file using `.env.example`:

```bash
cp .env.example .env
```

Run backend:

```bash
npm run dev
```

---

## 3. Frontend Setup

```bash
cd frontend
npm install
```

Create `.env` file using `.env.example`:

```bash
cp .env.example .env
```

Run frontend:

```bash
npm run dev
```

---

## 4. Run Project

- Backend: http://localhost:5000  
- Frontend: http://localhost:5173
